"CalcTape é uma calculadora de 4 operações e que armazena as contas anteriores como uma fita de calculadora antiga. O código utiliza estruturas como while, if, else, for para realizar o controle de fluxo."

Modo de usar
Compile o programa utilizando o comando javac Main.java
execute o programa utilizando o comando java main
siga as instruções do terminal
digite dois numeros, escolha uma operação
veja o resultado e o histórico das contas anteriores
digite exit para sair do programa
